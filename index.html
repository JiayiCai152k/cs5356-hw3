HTML Element Not Covered in Detail


Q: What HTML element did I use that we haven't covered in detail?

A: I used the <details> and <summary> elements. These allow users to toggle additional content without needing JavaScript.

Q: What did I learn about its behavior?

A: By default, the content inside <details> is hidden until the user clicks on <summary>. This is useful for organizing extra information in a collapsible format.

CSS Property Not Covered in Detail


Q: What CSS property did I use that we haven't covered in detail?

A: I used the filter property, which allows applying visual effects like blur, brightness, contrast, and grayscale.

Q: How did I use it?

A: I applied the filter: grayscale(100%); effect to my profile image when hovered over. This makes the image turn black & white smoothly. I also used transition to ensure a smooth visual effect.
